## Getting Started

### Prerequisites
To get started, run the following script which will install all the necessary prerequisites, including Bochs(Unless you already have it downloaded):

```sh
./Setup.sh
```

```sh
./Dev.sh
```

Additionally, you can install .NET by running:
```sh
./dotnet-install.sh
```
