## Getting Started

### Prerequisites
To get started with Puhaa-OS, run the following script which will install all the necessary prerequisites, including Bochs:

```sh
  ./Setup.sh
```

```sh
./Dev.sh
```

Additionally, you can install .NET by running:
```sh
./DotnetInstall.sh
```
